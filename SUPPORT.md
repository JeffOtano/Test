# Support

## Getting Help

For general usage questions:
- Open a GitHub issue using the appropriate template.

For security issues:
- Do not create a public issue.
- Use GitHub private vulnerability reporting:
  `https://github.com/JeffOtano/goodbye-shortcut/security/advisories/new`.

For code of conduct concerns:
- Contact maintainers via GitHub profile direct message, or open a `code-of-conduct` issue.

## What To Include

When requesting help, include:
- App version/commit SHA
- Node and npm versions
- Browser and OS
- Exact error message and stack trace
- Reproduction steps
